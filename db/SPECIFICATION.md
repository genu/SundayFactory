## Database Specification
### Overview
Songs are maintained in the `db` directory based on the following specifications. Each song is self contained in each own file. The name of the file is not part of the specifications, but for maintanability, the file name should be named identical to the name of the song in camel case format.

### Song information
````
{
  name: "Song Name"
  authors: ["Author1", "Author2"]
}
